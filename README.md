## What I'm building

**[HEARD 🏔️ ](https://github.com/luciobaiocchi/heard)**
An offline group-safety mesh for hikers: ESP32 + GPS + LoRa devices that keep
a group together where phones don't work. Born as my bachelor's thesis, now
open source, with a firmware-in-the-loop simulator that runs the real C++
protocol along real GPX tracks and replays it on 3D terrain in the browser.
